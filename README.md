# GL-E750-OWRT
Making Vanilla OpenWRT actually usable on the GL-iNet GL-E750

### Support
- [ ] Booting
- [ ] Charging (Not sure if this will be an issue)
- [ ] System Stats (Battery, Temp)
- [ ] Wi-Fi (Transmitting)
- [ ] Wi-Fi (Reciever Mode)
- [ ] Ethernet (LAN)
- [ ] Ethernet (WAN)
- [ ] Micro SD Slot
- [ ] LTE WWAN (Including SIM Slot)
- [ ] Display/MCU (Status, SSID, Signal, Battery)
